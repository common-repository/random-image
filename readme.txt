== Description ==

This plugin displays random images in the sidebar, posts, or any other location in WordPress. With this plugin user can delete uploaded image and even can mark it as Publish or Unpublish.

**Features**

* "No" loading time. the images are loaded dynamically one at a time from large group of image uploaded.
* Can be integrated in template files, posts and pages.
* No limitation in size at the time of uploading.
* Can fix image height only or width only or both while displaying image or you can also Leave the image as it is.


== Installation ==

1. Download "Random Image" plugin.
1. Open /wp-content/plugins folder at your web server.
1. Upload the folder "Random Image" there.
1. Goto Plugins page in your wordpress back office.
1. Activate "Random Image" plugin.
1. Go to Settings >> Random Image for modifying the settings.

Note: This plugin will require one time free registration.

== How to use it ==

* Go to Settings >> Random Image
* upload images.
* Adjust Advance Image Option.
* Go on widgets to display your random image on siderbar.
* use Post Tag to display random image  on post and page.
* use template Tag to display anywhere on your site.
